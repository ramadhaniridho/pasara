# Pasara

Market intelligence dashboard. Scraping harga + stok marketplace, insight AI, notifikasi otomatis.

## Stack

- **Frontend:** Next.js 14 + Tailwind + shadcn/ui
- **Backend:** Next.js API routes + Prisma
- **DB:** PostgreSQL (Docker local, Neon prod)
- **Scraper:** Puppeteer (TypeScript worker)
- **CI:** GitHub Actions
- **Deploy:** Vercel + Neon

## Dev

```bash
pnpm install
docker compose up -d
pnpm run dev
```
install pnpm using 
npm install -g pnpm
